# Dashboard Analítico de Vendas Globais

Projeto prático desenvolvido durante o Laboratório da Data Science Academy para exercitar conceitos de Visualização de Dados (Data Viz) e UX (User Experience).

## 📊 O Dashboard
![Visualização do Dashboard](Lab1.pdf)

## 🛠️ Objetivos de Negócio
O relatório foi estruturado para responder a perguntas críticas de vendas:
- Qual o valor total vendido (KPI principal)?
- Qual a média de desconto por subcategoria de produto?
- Quantas vendas foram realizadas por país considerando a prioridade de entrega?

## 🧠 Competências Desenvolvidas
- Construção de layout em Modo Escuro (Dark Mode).
- Organização dos elementos utilizando alinhamento em Grid.
- Criação de hierarquia visual para tomada de decisão gerencial rápida.
